# Reverse Camera
Basic Reverse Camera script for FiveM

## Preview
A Preview is available [here](https://youtu.be/E9OzArY3cQw)

## Features
- Button or Command Configurable
- Runs at 0.00ms
- Standalone

## Installation
Drag and drop into your resources folder and start in your server.cfg
